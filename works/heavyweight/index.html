HeavyWeight

Roles: Programming, Game Design

Platforms: Windows, Mac OS, GNU/Linux

Download: GGJ Website

HeavyWeight was made during the Global Game Jam event in 2014 using the Unity engine. The design was made together with a friend of mine who also did the visuals.

The game came out as an experimental idea where the player is encouraged to question the way how the games give us orders, making a connection to the real life.

The game is published under the WTFPL and the sources can be found in the download link above.
