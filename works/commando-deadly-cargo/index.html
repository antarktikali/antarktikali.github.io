Commando: Deadly Cargo

Roles: Programming

Platform: Apple iOS

Download: iTunes Link

Company: Fenris Entertainment

Commando: Deadly Cargo is a game available for the iPhone and iPad devices. It was developed using Unity. The game was initially being sold for 0.99$, but now you can get it for free of charge.

I was involved in the project during the last month of the development. My work included the implementation of the achievement system, various iPhone/iPad related optimisations and the user interface.
